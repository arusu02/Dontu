# DONTUCONSTRUCTION - Website

Professional construction company website with premium design.

## Project Structure

```
dontu-construction/
├── index.html          Main HTML file
├── css/
│   └── style.css       All styles
├── js/
│   └── script.js       All JavaScript
├── images/             Put your images here
└── README.md           This file
```

## Color Palette

- Primary Gold: #C9A961
- Secondary Gold: #D4AF37
- Dark Charcoal: #1A1A1D
- Medium Charcoal: #2D2D30
- Cream: #F5F5DC

## How to Replace Images

1. Put your images in the images folder
2. In index.html find: `<div class="service-image">IMG 1</div>`
3. Replace with: `<img src="images/your-image.jpg" alt="Description" style="width:100%; height:100%; object-fit:cover;">`

## Features

- Fully responsive (Desktop, Tablet, Mobile)
- Premium gold and charcoal color scheme
- Smooth scroll animations
- Sticky navigation
- Hover effects

## How to Use

1. Open index.html in your browser
2. Customize the content
3. Add your images
4. Upload to your web hosting
